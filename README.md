# Preço Justo (MVP)

Projeto para comparar preços de supermercados de forma colaborativa e sem login.

- Backend: Node.js + Express (arquivo principal: backend\servidor.js)
- Banco: MySQL (schema em database\schema.sql)
- Frontend: HTML, Bootstrap, JS (frontend/)

Passos rápidos:
1. Criar banco MySQL com database\schema.sql
2. Copiar backend\.env.example -> backend\.env e ajustar
3. No terminal:
   cd backend
   npm install
   npm start
4. Abrir http://localhost:3000
